# WTTT

Workshop Technician Time Tracking (WTTT) is an in-house workshop operations system designed and implemented by **Noha Shekib**.

## Run (local)

1. Install dependencies:
   - `npm install`
2. Configure environment:
   - Create a `.env` file (see `src/database/connection.js` for required variables)
3. Start server:
   - `npm start`

## Notes

- The application serves role dashboards from the project root (`index.html`, `dashboard-*.html`) and API endpoints from `server.js` / `src/`.
